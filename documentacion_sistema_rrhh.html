Sistema RRHH

Documentación Completa del Sistema de Recursos Humanos

Django 5.2.5 Python 3.11.9 SQLite Bootstrap

Resumen Ejecutivo

El Sistema RRHH es una aplicación web completa desarrollada en Django que gestiona empleados, solicitudes de vacaciones, nuevos colaboradores y jerarquías organizacionales. El sistema implementa un robusto sistema de permisos basado en roles y jerarquías, con interfaces modernas y responsivas.

Gestión de Empleados

Sistema completo de CRUD para empleados con validaciones y campos específicos como DNI.

Solicitudes de Vacaciones

Flujo completo de solicitudes con aprobaciones por jerarquía y control de días disponibles.

Nuevos Colaboradores

Proceso de solicitud y aprobación para incorporación de nuevos empleados al equipo.

Sistema de Permisos

Control granular de acceso basado en roles y jerarquías organizacionales.

Jerarquía Organizacional

El sistema implementa una jerarquía de 8 niveles que determina permisos y flujos de aprobación:

1
Director - Máximo nivel jerárquico
2
Gerente - Gestión de áreas completas
3
Sub-gerente - Apoyo a la gerencia
4
Jefe - Liderazgo de equipos
5
Supervisor - Supervisión directa
6
Coordinador - Coordinación de procesos
7
Asistente - Apoyo especializado
8
Auxiliar - Apoyo operativo

Creación Automática de Empleados

Una de las características más innovadoras del sistema es la creación automática de empleados cuando RRHH aprueba solicitudes de nuevos colaboradores.

Proceso Automatizado

1. Aprobación por RRHH

RRHH revisa y aprueba la solicitud de nuevo colaborador con todos los datos requeridos.

2. Creación de Usuario Django

Se crea automáticamente un usuario en el sistema con username formato nombre.apellido (ej: juan.perez).

3. Generación de Contraseña Temporal

Se genera una contraseña segura de 12 caracteres aleatorios para el primer acceso.

4. Creación de Empleado

Se crea el perfil de empleado con todos los datos de la solicitud, incluyendo jerarquía y manager asignado.

5. Entrega de Credenciales

Se presenta una interfaz especial con las credenciales para que RRHH las entregue al nuevo empleado.

Seguridad y Validaciones

  • Verificación de DNI único antes de crear el usuario
  • Contraseñas temporales generadas con algoritmos seguros
  • Username único en formato nombre.apellido con normalización de acentos
  • Resolución automática de conflictos de username agregando números secuenciales
  • Mapeo automático de denominación de puesto a jerarquía organizacional
  • Asignación inteligente de gerencia basada en el área
  • Transacciones atómicas para garantizar consistencia de datos
  • Manejo de errores con rollback automático
